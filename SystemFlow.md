 System User Flow (English Version)
1. Student

View personal attendance records.

View lecture timetable.

Receive notifications when absence count reaches 2 or 3.

2. Instructor

View own lecture schedule.

Take attendance for assigned lectures.

3. Academic Advisor

View lecture schedules for all groups supervised.

View attendance reports for students in supervised groups.

4. Level Registrar

Add / Edit lecture and group schedules for all courses in their level.

View student lists for each group in their supervised level.

5. Student Affairs

View all lecture schedules.

Access all student lists across all levels.

View each studentâ€™s assigned group.

Track registered student count and their current status (active, withdrawn, etc.).

View attendance reports for all students across levels.

Manage sub-data / settings (Settings page).

6. General Registrar & General Coordinator

Have all permissions of Student Affairs.

Additionally, can edit any data or override lower-level user actions.

| **Role** | **Main Responsibilities** | **Features / Permissions** |
|-----------|---------------------------|-----------------------------|
| **Student** | Track attendance and view schedule | - View personal attendance record<br>- View class schedule<br>- Receive notifications when absences reach 2 or 3 |
| **Instructor** | Manage lecture attendance | - View assigned lecture schedules<br>- Record attendance for each lecture |
| **Academic Advisor** | Monitor groups and attendance | - View lecture schedules for supervised groups<br>- View attendance reports for those groups |
| **Level Registrar** | Manage lecture and group schedules | - Add / edit schedules for all lectures and groups<br>- View and manage student lists per group and level |
| **Student Affairs** | Full academic and attendance management | - View lecture schedules for all levels<br>- Access full student lists and group assignments<br>- View number of registered students and their statuses<br>- View attendance reports for all students<br>- Manage secondary data (Settings Page) |
| **General Registrar / Coordinator** | High-level management and supervision | - All permissions of Student Affairs<br>- Additional authority to edit and update data globally |
